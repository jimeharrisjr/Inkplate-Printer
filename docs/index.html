Inkplate E-Ink Printer

Turn your Inkplate into a wireless network printer. Print from any app on your Mac or Windows PC.

What You Need

  • Inkplate 6 (800×600, grayscale) or Inkplate 6 Color (600×448, 7-color)
  • Micro SD card — FAT32 formatted, any size (4GB+ recommended)
  • Arduino IDE 1.8 or later (or Arduino IDE 2.x)
  • USB cable — micro-USB or USB-C depending on your Inkplate model
  • WiFi network — 2.4 GHz (the ESP32 does not support 5 GHz)

Step 1 — Install Arduino & Inkplate Board Package

Install Arduino IDE

Download from arduino.cc and install it.

Add the Inkplate board URL

Open Arduino IDE. Go to File → Preferences (or Arduino → Settings on Mac).

In the Additional Board Manager URLs field, add:

https://github.com/SolderedElectronics/Dasduino-Board-Definitions-for-Arduino-IDE/raw/master/package_Dasduino_Boards_index.json

Install the Inkplate board package

Go to Tools → Board → Board Manager. Search for Inkplate and install Inkplate Boards.

Install the Inkplate library

Go to Sketch → Include Library → Manage Libraries. Search for InkplateLibrary and install it.

Step 2 — Configure for Your Hardware

Open the project folder in Arduino IDE by opening inkplate_print.ino.

Open config.h and uncomment the line matching your Inkplate model:

For the Inkplate 6 (grayscale, 800×600), uncomment this line:

#define INKPLATE_MODEL_MONO    // Inkplate 6 (800x600, 3-bit grayscale)

And make sure the Color line is commented out:

// #define INKPLATE_MODEL_COLOR   // Inkplate 6 Color (600x448, 7-color)

This is the default — if both lines are commented, the Inkplate 6 mono is selected automatically.

For the Inkplate 6 Color (7-color, 600×448), uncomment this line:

#define INKPLATE_MODEL_COLOR   // Inkplate 6 Color (600x448, 7-color)

And make sure the Mono line is commented out:

// #define INKPLATE_MODEL_MONO    // Inkplate 6 (800x600, 3-bit grayscale)

Optional: Change the printer name

The printer appears on your network as Inkplate-printer. To change this, edit:

#define PRINTER_NAME  "Inkplate-printer"

Step 3 — Prepare the SD Card

Insert a micro SD card into your computer (using an adapter if needed).

Format it as FAT32. On Mac: open Disk Utility, select the card, click Erase, choose format MS-DOS (FAT). On Windows: right-click the drive in File Explorer and choose Format → FAT32.

Insert the SD card into the Inkplate's micro SD card slot.

The Inkplate will create a /print/ folder on the SD card automatically. You don't need to create any files or folders.

Step 4 — Upload the Sketch

Connect the Inkplate to your computer via USB.

Select the correct board

Go to Tools → Board and select:

ModelBoard selection
Inkplate 6e-radionica.com Inkplate6
Inkplate 6 ColorSoldered Inkplate 6COLOR

Select the serial port

Go to Tools → Port and select the port for your Inkplate (usually /dev/cu.usbserial-... on Mac or COM3+ on Windows).

Upload

Click the Upload button (right arrow icon). Wait for the compile and upload to finish.

If the upload fails with a connection error, try holding the BOOT button on the Inkplate while pressing RESET, then release BOOT and click Upload again.

Step 5 — Connect to WiFi

After uploading, the Inkplate boots and shows a setup screen:

On your phone or computer, connect to the WiFi network Inkplate-Setup (no password).

A captive portal page should open automatically. If not, open a browser and go to http://192.168.4.1.

Enter your WiFi network name (SSID) and password, then click Connect.

The Inkplate restarts and connects to your network. It displays its IP address on the screen.

The WiFi credentials are saved in the Inkplate's non-volatile storage. You only need to do this once. To change networks later, reset the Inkplate's stored credentials by reflashing the sketch.

Step 6 — Add the Printer to Your Computer

Mac (recommended method)

Open Terminal and run:

lpadmin -p Inkplate -v ipp://Inkplate-printer.local:631/ipp/print -m everywhere
cupsenable Inkplate

The printer now appears in System Settings → Printers & Scanners and in all applications' Print dialogs.

You can also try adding via System Settings → Printers & Scanners → Add Printer. The Inkplate appears as Inkplate-printer under Bonjour. If the “Use” dropdown says “Choose a Driver”, use the Terminal method above instead.

Windows

Go to Settings → Devices → Printers & scanners → Add a printer. The Inkplate should appear in the list. Select it and click Add device.

If it doesn't appear, click The printer I want isn't listed, choose Add a printer using a TCP/IP address, and enter:

  • Hostname: Inkplate-printer.local
  • Port: 631
  • Device type: IPP

Step 7 — Print Something

Open any document, web page, or email, and print it. Select Inkplate as the printer.

Paper size tips

The Inkplate advertises a custom page size (~3.6" × 4.8") that matches the physical display. Most apps pick this up automatically. For best results:

  • Web browsers (Safari, Chrome): Usually work without any changes. The page is automatically scaled to fit.
  • Microsoft Word: Go to Layout → Size → More Paper Sizes and set a custom size of 3.61" × 4.82" (or 91.8mm × 122.4mm). Set margins to 0.3" or smaller.
  • Plain text (Terminal): echo "Hello" | lp -d Inkplate

Using the Inkplate

Status bar

The bottom of the display shows: Doc 1/3   Page 2/5   WiFi

Storage

Documents are stored on the SD card in /print/doc_NNNN/ folders. When the SD card fills up, the oldest document is automatically deleted to make room for new prints.

Troubleshooting

Printer doesn't appear on the network

  • Check the Inkplate screen — does it show an IP address? If it shows “WiFi Setup”, the WiFi credentials need to be entered.
  • Make sure your computer and the Inkplate are on the same WiFi network.
  • Try: dns-sd -B _ipp._tcp in Terminal (Mac) to check if the printer is broadcasting.

Print jobs fail or hang

  • Check that the SD card is inserted and formatted as FAT32.
  • Open Arduino IDE's Serial Monitor (115200 baud) to see debug output from the Inkplate.
  • Re-enable the printer if it got disabled: cupsenable Inkplate

Text is too small

  • The printer's page size is ~3.6" × 4.8". If an app prints to a larger page (Letter, A4), the content is scaled down to fit. Set the paper size in the app to match the Inkplate's custom size.
  • In web browsers, you can reduce the scale percentage in the Print dialog.

Display shows a blank page

  • Some text filters (like the macOS plain text filter via lp) may produce blank pages for custom page sizes. Try printing from a real app (Safari, TextEdit) instead.

Upload fails

  • Try a lower upload speed: Tools → Upload Speed → 115200.
  • Hold the BOOT button, press RESET, release BOOT, then click Upload.
  • Try a different USB cable (some are charge-only).
